Le site Multiplicorne

Un site pour réviser ses tables de multiplication, mais kawaï !

Réalisé avec JS, PHP et MySQL ; on peut réviser les tables une à une, dans l'ordre ou le désordre.
Ou alors, c'est le "marathon", toutes les tables dans le désordre !
Inclus un espace membre avec accès par mot de passe. Les temps réalisés sont conservés ce qui permet de les comparer et de voir les progrès réalisés.
Ma fille adore ça.
